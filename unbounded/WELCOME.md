## The Two Gardens

### The Termite Project — Practical Science

**What it is:** Real laboratory work — isolating and culturing bacteria from the guts of *Macrotermes gilvus* termites collected in Southern Luzon, Philippines. The goal was threefold: convert agricultural waste into ethanol, develop natural pest control formulations, and establish captive termite farming for protein and research.

Three interconnected research initiatives, one lab, one team, one vision:

**1. Biofuel Research:** 7 bacterial strains (TF-001 through TF-007) isolated from termite gut. A C-5 enzyme cocktail achieving 89.4% cellulose conversion. Co-culture fermentation (TF-001+TF-006) producing 10.8 g/L ethanol — 95% of theoretical yield. Scale-up to 5L bioreactor fed-batch reaching 12.5 g/L. A breakeven price of P23.23/L at 300,000 L annual production.

**2. Natural Pest Control:** 4 formulations tested with 88-100% efficacy against target pests. An essential oil blend achieving 100% mortality at just 250 uL/L. A US$1B+ annual market opportunity in Southeast Asia alone. Manufacturing guides, safety data sheets, and certification roadmaps already written.

**3. Termite Farming:** 3 *M. gilvus* colonies established with 86% growth rate. Protein content of 40-60% by dry weight — competitive with conventional protein sources. Thailand benchmark model producing 3,700 tons/year at ~3.4 USD/kg. Seven-part implementation plan covering facility design, colony management, processing, quality control, and distribution.

**Key numbers at a glance:**

| Metric | Value |
|--------|-------|
| Bacterial strains isolated | 7 (TF-001 to TF-007) |
| Cellulose conversion rate | 89.4% (C-5 enzyme cocktail) |
| Peak ethanol production | 10.8 g/L (co-culture) |
| 5L bioreactor fed-batch | 12.5 g/L ethanol |

| — Biofuel | P260M |
| — Pest control | P37M |
| — Termite farming | P22M |

**Status:** Phase 1 complete (January — June 2026). Isolation, characterization, and proof-of-concept done. The science works. Ready for scale-up — larger bioreactors, field trials, FDA/DENR certification, manufacturing partnerships, and market entry.

**Located at:** `../termite-projects/` — a complete research repository with 50+ files covering lab notebooks, experiment data (4 CSV files with full numerical results), strain catalogs, enzyme characterizations, manufacturing guides, safety protocols, investor materials, pitch decks, financial projections, and implementation timelines.

### Step 4: Explore the Data -> `site/dashboard.html`

*The evidence — real experiments.*

This dashboard loads real CSV files from the termite project biofuel research and renders them as interactive charts. You can see the cellulase assay data, fermentation curves, glucose calibration regressions, and agricultural waste test results. The data is not simulated. These are real measurements from real lab work.

This is where the vision meets the ground. While UNBOUNDED dreams of the future of media, the termite project is in a lab in the Philippines, measuring real things, getting real results. 7 bacterial strains. 89.4% conversion. 10.8 g/L ethanol. 4 formulations. 3 colonies. The dashboard proves it.

The CSV files that power this dashboard live in the termite project repository at `../termite-projects/biofuel-research/data/`. You can inspect them directly, run your own analysis, or contribute new experiments.

### Step 5: Run the Gateway -> `src/gateway.py`

*The CLI — type, explore.*

The simplest way: run `python run.py` from the project root. This opens a menu where you can:
- [1] Launch the CLI Gateway
- [2] Start a web server on port 8080
- [3] Open the portal in your browser
- [4] Open the data dashboard
- [5] Open the landing page
- [6] Open the project folder

Or run `python src/gateway.py` directly for the full CLI experience — a 695-line Python program with 9 menu options covering: termite project summary, experimental data analysis (with linear regression, CSV parsing, and strain rankings), lab notebook browsing, UNBOUNDED vision exploration, calls to action, investor materials, cross-project connections, portal launcher, and help/about.

The gateway is the bridge. It is where the two projects — soil and screen — become one interface. It is also a good example of the code style and structure if you are a developer looking to contribute.
