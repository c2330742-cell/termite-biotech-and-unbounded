Manufacturing Guide for Natural Termite Control Products

1. Facility Requirements

1.1 Production Space

Minimum Requirements:
- Total area: 100-200 square meters
- Mixing room: 30-50 square meters
- Storage area: 30-50 square meters
- Packaging area: 20-30 square meters
- Quality control lab: 10-20 square meters
- Office space: 10-20 square meters

Climate Control:
- Temperature: 20-30°C
- Humidity: 40-60%
- Ventilation: Mechanical exhaust system
- Lighting: 300-500 lux

1.2 Equipment List

Mixing Equipment:
- Stainless steel mixing tanks (500-1000L)
- High-shear mixer (1-5 HP)
- Low-speed paddle mixer
- Heating/cooling system
- Temperature and pH monitoring

Filling Equipment:
- Liquid filling machine (manual or semi-automatic)
- Granule filling machine
- Capping machine
- Labeling machine

Packaging Equipment:
- Heat sealer
- Shrink wrap machine
- Cardboard box former
- Palletizer (optional)

Quality Control Lab:
- pH meter
- Viscosity meter
- Microscope
- Balance
- Incubator

1.3 Raw Materials Storage

Temperature-Controlled Storage:
- Refrigerated storage (4-10°C): For nematodes and fungal cultures
- Cool storage (15-25°C): For essential oils and botanical extracts
- Room temperature storage (20-30°C): For dry materials

Safety Requirements:
- Fire extinguishers
- Spill containment
- Ventilation system
- Secondary containment for liquids

2. Production Procedures

2.1 Botanical Spray Production

Batch Size: 500 liters

Procedure:
1. Prepare water in mixing tank
2. Heat to 40-50°C if required
3. Add emulsifier and mix until dissolved
4. Add active ingredient (neem oil, citrus oil)
5. High-shear mix for 15-30 minutes
6. Cool to room temperature
7. Add preservative if required
8. Adjust pH to 6.0-7.5
9. Test sample for quality
10. Filter if necessary
11. Fill into containers
12. Label and package

Quality Checkpoints:
- pH measurement
- Viscosity check
- Visual inspection
- Active ingredient content

2.2 Granular Product Production

Batch Size: 200 kg

Procedure:
1. Mix dry ingredients in ribbon blender
2. Add liquid binders gradually
3. Mix until uniform consistency
4. Granulate using extruder or pan granulator
5. Dry to 5-10% moisture content
6. Screen to remove fines and oversize
7. Apply surface treatment if required
8. Package in moisture-proof containers

Quality Checkpoints:
- Particle size distribution
- Moisture content
- Bulk density
- Active ingredient content

2.3 Bait Station Production

Batch Size: 1000 stations

Procedure:
1. Prepare bait formulation
2. Fill bait matrix into stations
3. Attach stations to backing plate
4. Package in moisture-proof bags
5. Label with instructions and expiration

Quality Checkpoints:
- Bait weight per station
- Moisture content
- Active ingredient content
- Package integrity

- Moisture-proof containers

- Shrink-wrapped packages

2. Seal bags
3. Apply labels

10. Emergency Procedures

10.1 Spill Response

Small Spills:
1. Contain spill with absorbent material
2. Clean up immediately
3. Dispose of waste properly
4. Document incident

Large Spills:
1. Evacuate area
2. Notify emergency response
3. Contain spill if safe
4. Clean up with appropriate equipment
5. Document incident

10.2 Equipment Failure

Immediate Response:
1. Stop production
2. Secure equipment
3. Notify supervisor
4. Document incident

Recovery:
1. Assess damage
2. Repair or replace equipment
3. Verify safety
4. Resume production

10.3 Medical Emergencies

First Aid Response:
1. Assess situation
2. Provide first aid
3. Call for medical help
4. Document incident

Prevention:
1. Safety training
2. First aid kits
3. Emergency contacts
4. Regular drills
