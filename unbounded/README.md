# UNBOUNDED

**Information is entertainment. Creation is consumption. Everyone is a broadcaster.**

---

## The Revolution

Television is dead. Scripts are cages. The old media fed you passive content designed to harvest your attention and sell it to the highest bidder. You were the product, not the participant.

**UNBOUNDED** is the antidote.

A new media platform where:
- **Every viewer is also a creator** — real-time tools to generate, modify, and shape content as it happens
- **Every session is unique** — AI-facilitated, procedurally generated, co-created with you
- **Deep knowledge IS the entertainment** — learning a skill, debating an idea, building something real — all are as gripping as the best drama
- **No gatekeepers** — the underground becomes mainstream. The unwelcome becomes welcome. Suppressed knowledge finds light.
- **No surveillance** — your attention is yours. No ads, no profiling, no algorithmic manipulation.

## How to Experience

**The simplest path:**
```
python run.py
```
Then select [2] to start the web server, or [1] for the CLI gateway.

**Or directly:**
- `site/portal.html` — the interactive portal (open in browser)
- `site/dashboard.html` — the data dashboard (open in browser)
- `src/gateway.py` — the CLI experience (run with python)
