# Presentations & Investor Materials

Created for a Shark Tank / VC funding round. All documents reference actual experimental data from Phase 1 (Jan–Jun 2026).

| File | Description |
|------|-------------|
| `SHARK-TANK-PITCH-SCRIPT.md` | Full 8–10 minute spoken pitch script with stage directions — dramatic "rise of the new, fall of the corrupted" narrative |
| `PITCH-DECK-SLIDES.md` | 20-slide deck: visual descriptions, on-screen text, speaker notes, transitions |
| `INVESTOR-QA-PREP.md` | 20 tough investor questions with data-backed answers, risk acknowledgments, and follow-ups |
| `PRESS-RELEASE.md` | Breakthrough announcement press release with quotes from PI, independent expert, and DOST official |

## How to Use

1. **Practice**: Read the pitch script aloud with a timer (target 8–10 minutes)
2. **Deck**: Build slides from `PITCH-DECK-SLIDES.md` — each slide has the visual spec and text
3. **Prep**: Study `INVESTOR-QA-PREP.md` before any investor meeting — hardest questions first
4. **Announce**: Use `PRESS-RELEASE.md` after closing the round

All numbers are internally consistent with the data in `biofuel-research/data/`, `CROSS-VERIFICATION.md`, and `MASTER-PROJECT-SUMMARY.md`.
