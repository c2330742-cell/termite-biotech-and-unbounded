# Experiment Records

| File | Description |
|------|-------------|
| `experiment-log.md` | Quick-reference log of all 10 experiments with key results |
| `LAB-NOTEBOOK.md` | Formal lab notebook — full write-ups with raw data, calculations, observations, and conclusions (1,513 lines, 10 entries) |
| `EXPERIMENT-REGISTRY.csv` | Machine-readable index of all experiments for data analysis scripts |

## How to Use

- **Quick reference**: Read `experiment-log.md` for an overview of each experiment
- **Detailed review**: Open `LAB-NOTEBOOK.md` for the full scientific record
- **Data analysis**: The registry CSV maps entries to CSV data files in `../../data/`
- **Cross-referencing**: Check `CROSS-VERIFICATION.md` at the project root for literature comparison

## Lab Notebook Conventions

- Each entry is numbered sequentially and dated
- Raw data is recorded before calculations
- All measurements include error estimates (±SD)
- Deviations from protocol are noted in Observations sections
- Each entry ends with a signature block for PI review
