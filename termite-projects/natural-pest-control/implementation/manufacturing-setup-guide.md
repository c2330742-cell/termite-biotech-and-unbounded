## MANUFACTURING PROCEDURES

### Standard Operating Procedure: Batch Production

**SOP Number:** NTPC-MFG-001
**Effective Date:** Month 6
**Review Date:** Month 18

#### 1. Pre-Production
1.1 Review production schedule
1.2 Verify raw material availability
1.3 Check equipment status
1.4 Don appropriate PPE
1.5 Clean and sanitize production area

#### 2. Material Preparation
2.1 Weigh raw materials per batch record
2.2 Verify weights against specifications
2.3 Stage materials in order of addition
2.4 Record lot numbers for traceability

#### 3. Mixing Process
3.1 Add Phase A ingredients to mixing tank
3.2 Start mixer at 100 RPM
3.3 Heat to 25°C if required
3.4 Add Phase B ingredients
3.5 Increase speed to 200 RPM
3.6 Add Phase C ingredients slowly
3.7 Mix for 30 minutes
3.8 Add preservative
3.9 Mix for 15 minutes

#### 4. Quality Check
4.1 Sample product (100 mL)
4.2 Test pH (target: 5.5-6.5)
4.3 Test viscosity (target: 10-50 cP)
4.4 Check appearance (clear to amber)
4.5 Record results in batch record
4.6 Release or hold for rework

#### 5. Filling Process
5.1 Transfer product to filling tank
5.2 Calibrate filling equipment
5.3 Run first-piece inspection
5.4 Begin production run
5.5 Monitor fill weights every 15 minutes
5.6 Apply labels and caps
5.7 Induction seal containers

#### 6. Packaging
6.1 Inspect finished containers
6.2 Pack into cartons
6.3 Shrink-wrap cartons
6.4 Palletize and stretch-wrap
6.5 Move to finished goods warehouse

#### 7. Documentation
7.1 Complete batch record
7.2 Update inventory system
7.3 File quality control records
7.4 Archive documentation
